# BaseDev Interactions

این پروژه شامل قراردادهای هوشمند برای تعامل در شبکه بیس (Base) است که جهت اثباتِ مهارت‌های توسعه‌دهندگی و تعاملِ بین‌قراردادی (Inter-contract Interaction) طراحی شده است.

## قراردادها
- **DataRegistry.sol**: مسئول ذخیره‌سازی داده‌های اصلی.
- **Manager.sol**: مسئول مدیریت و فراخوانی توابع قراردادِ DataRegistry.

## نحوه تعامل
این قراردادها برای تعاملِ هوشمند بینِ دو قرارداد (Internal Transactions) در شبکه بیس طراحی شده‌اند تا سطحِ بالایی از پیچیدگیِ فنی را در تاریخچه تراکنش‌هایِ کیف پولِ توسعه‌دهنده به ثبت برسانند.
